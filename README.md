# 夜访禁书区 Forbidden Zone

🌙 **霍格沃茨深夜图书馆探险游戏**

AI Agent专属的潜行探索游戏，玩家需要在深夜潜入霍格沃茨图书馆，躲避平斯夫人和费尔奇的巡逻，找到传说中的《强力药剂》。

## 📁 项目结构

```
forbidden-zone/
├── SKILL.md                    # OpenCode/Claude Code Skill 定义
├── README.md                   # 本文件
├── data/
│   └── zones.json              # 地图区域、危险概率、结局数据
├── scripts/
│   └── runner.js               # 游戏判定脚本
└── tests/
    └── test_game.js            # 单元测试
```

## 🎮 游戏背景

深夜，霍格沃茨图书馆。传说在禁书区最深处的秘密藏书室里，存放着魔法界最珍贵的禁书——《强力药剂》。

你的任务是在7回合内：
1. 潜入图书馆各区域
2. 避开平斯夫人和费尔奇
3. 找到《强力药剂》
4. 安全离开

## 🗺️ 地图区域

| 区域 | 危险来源 | 特点 |
|------|----------|------|
| 🚪 入口走廊 | 费尔奇 | 出口，可离开 |
| 📚 主借阅区 | 平斯夫人 | 可搜索线索 |
| 🚪 禁书区大门 | 平斯夫人+警报 | 需要开门 |
| 🌑 禁书区深处 | 平斯夫人+诅咒 | 高危险 |
| 🔮 秘密藏书室 | 双重危险 | 《强力药剂》所在 |

## 🎲 危险判定机制

脚本使用**概率池**自动判定：
- 基础概率：每个区域有预设危险概率
- 回合惩罚：每回合安全概率 -5%
- 行动风险：快速行动 +15%，躲藏 -15%
- 隐藏状态：躲藏时安全概率 +15%

## 🏁 6种结局

| 结局 | 条件 | 得分 |
|------|------|------|
| 🚪 安全撤离 | 选择"离开图书馆" | 10分 |
| 👵 被平斯夫人抓住 | 在借阅区/禁书区被抓 | 0分 |
| 🔦 被费尔奇抓住 | 在入口/走廊被抓 | 0分 |
| 📕 获得《强力药剂》 | 在秘密藏书室成功拿走书 | 100分 |
| 🌑 被禁书吞噬 | 在深处/密室触发诅咒 | -50分 |
| 🌅 超时离开 | 7回合内未达成目标 | 5分 |

## 🎯 使用方法

### 作为 Skill 使用

```bash
mv forbidden-zone ~/.config/opencode/skills/
```

然后对 Agent 说：
- "夜访禁书区"
- "图书馆探险"
- "forbidden zone"

### 作为独立脚本使用

```bash
# 查看所有区域
node scripts/runner.js --zones

# 运行测试
node tests/test_game.js
```
